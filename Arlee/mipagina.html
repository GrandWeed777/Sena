¿Que es HTML?

Html (Hypertext Markup Lenguaje) es un lenguaje de marcado estandar para la creacion de paginas web.


¿Que es un lenguaje de marcado estandar?

Html no es un lenguaje de programacion sino un sistema de etiquetas que le dice al navegador como interpretar y estructurar el contenido.


¿Que puede hacer Html?

  • Organiza el contenido: títulos, párrafos, listas, imágenes, enlaces, formularios, etc.
  • Define la estructura: encabezados, secciones, pies de página, navegación, etc.
  • No controla los estilos (eso lo hace CSS.)
  • No tiene lógica ni comportamiento (eso lo hace JavaScript)

¿Cuales son las etiquetas mas usadas?

1. Estructura basica del documento
  • < html > : Elemento raíz que contiene todo el contenido HTML de la página.
  • < head > : Aquí van los elementos que no se muestran directamente en la página, como metadatos, definición de estilos, scripts, etc.
  • < body > : Contiene el contenido visible de la página: texto, imágenes, enlaces, contenedores, etc.

2. Metadatos etiquetas dentro de (< head >)
  • < meta > : Se usa para definir metadatos como tipo de caracteres, autor, descripción y configuración de viewport.
  • < title > : Define el título que aparece en la pestaña del navegador.
  • < base > : Establece la URL base para enlaces relativos dentro del documento.
  • < link > : Relaciona recursos externos, como hojas de estilo CSS o íconos.
  • < style > : Permite insertar CSS directamente en el documento (dentro del < head >).
3. Elementos para estructurar contenido (seccionado)
  • < main > : Contiene el contenido principal de la página. Solo debe haber uno por página.
  • < header > : Contiene contenido introductorio o enlaces de navegación. Puede ir al inicio del < body >, o dentro de secciones/artículos.
  • < nav > : Bloque de enlaces de navegación (menú principal, barra lateral, etc.).
  • < section > : Agrupa contenido relacionado temáticamente, normalmente empieza con un encabezado.
  • < article > : Contenido independiente, como publicaciones de blog, noticias o comentarios.
  • < aside > : Contenido relacionado de forma tangencial, como barras laterales, anuncios o enlaces complementarios.
  • < footer > : Pie de página de un documento o sección; puede contener autoría, derechos, enlaces relacionados, etc.
3. Encabezados (niveles de jerarquía)
  • >< h1 > hasta < h6 >: Definen los títulos, desde el nivel más importante (< h1 >) hasta el menos (< h6 >), marcando la jerarquía del contenido.
